# Grant Rinehart — Financial Analysis Website

A personal financial analysis site comparing Google (GOOGL), Tesla (TSLA), and Nvidia (NVDA) across FY2023–FY2025.

Built as a student research project. Features interactive charts, company breakdowns, profit margin comparisons, and a full report card.

## What's inside
- Revenue and net profit trends for all three companies
- Side-by-side comparison table with FY2025 ranges
- Final report card grades
- All Reports section linking to full written analysis

## Disclaimer
All data is sourced from publicly available information. FY2025 figures are reported as ranges. **This is not financial advice.** Always consult a licensed financial professional before making investment decisions.
